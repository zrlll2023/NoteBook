### 教学方式说明

- 每个阶段我会先问你问题，**引导你自己想出方案**，再带你实现
- 涉及 Spring 核心原理（**IOC / AOP / 自动装配**）时，我会在自然出现的地方展开讲，不会单独抽出来背概念
- 每写完一个模块，我会问你「能解释一下这段代码在做什么吗？」—— 这是为了让你能在面试中真正说得出来

### 码上记 · 程序员知识库平台（后端）

- **技术栈**：Spring Boot 3、MyBatis-Plus、Redis、Docker、MySQL、JWT、Knife4j
    
- **开源地址**：[memos-app (GitHub)](https://github.com/zrlll2023/memos-app "null")
    
- **项目描述**： 基于 Spring Boot 3 独立构建的一套高性能 RESTful API，提供用户注册/登录、文章管理、分类标签及评论等核心知识库功能。
    
- **核心工作**：
    
    - **核心功能**：实现基于 JWT + 拦截器的无状态安全鉴权方案；用户登出时通过将 Token 存入 **Redis 黑名单** 实现安全下线。
        
    - **动态查询**：巧妙使用 MyBatis-Plus 的 `LambdaQueryWrapper` 实现多条件动态组合查询与分页，大幅简化重复 SQL 编写并保障查询性能。
        
    - **性能优化**：集成 **Redis** 缓存热门文章列表（设置过期时间 TTL 为 10 分钟），并使用 `INCR` 命令实现高并发场景下的浏览量统计，有效降低底层 MySQL 数据库的读写压力。
        
    - **容器化部署**：编写 `Dockerfile` 与 `docker-compose.yml` 配置文件，实现应用服务、MySQL 以及 Redis 的一键本地及云端容器化部署。
        
    - **接口文档**：集成 **Knife4j** 自动生成实时接口文档，高效完成 15+ 个核心接口的设计、开发与前后端联调。
