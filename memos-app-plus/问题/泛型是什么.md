别急,是我「泛型」这个词甩得太快了,先把这个词钉死,你
  立刻就懂了。

  【知识点定位】 Java > 泛型(Generics)> 类型参数
  【标准定义】 泛型就是写在尖括号 <>
  里的那个类型。它的作用是告诉编译器「这个容器里装的是
  什么类型的东西」。

  拿你自己的代码看,一眼就懂:

  Result<Void>              // 尖括号里是 Void  → 这个
  Result 里不装数据
  Result<List<Category>>    // 尖括号里是
  List<Category> → 这个 Result 里装一个分类列表

  我说「泛型填什么」=「Result 后面那个尖括号 <>
  里写哪个类型」。就这么简单,不是什么高深东西。
