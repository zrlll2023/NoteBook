注解: @RestController
  作用: 标在类上,声明「这是个处理 HTTP
    请求的控制器,返回值直接当响应数据(JSON)」
  你在哪用过: ArticleController 类头

---

  注解: @RequestMapping("/api/article")
  作用: 标在类上,这个类所有接口的公共 URL 前缀
  你在哪用过: 类头

---

  注解: @PostMapping / @GetMapping / @DeleteMapping /
    @PutMapping
  作用: 标在方法上,绑定 HTTP
    动词(增/查/删/改),括号里可加子路径
  你在哪用过: 三个你都用过

---

  注解: @PathVariable
  作用: 标在参数上,从 URL 路径里取值。如 /{id} 配
    @PathVariable Long id
  你在哪用过: 删除、查详情

---

  注解: @RequestBody
  作用: 标在参数上,把请求体里的 JSON 转成一个 Java
  对象
  你在哪用过: 你新增分类/标签时用过 @RequestBody
