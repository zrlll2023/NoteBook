| **数据大类**    | **类型名称**        | **占用空间/特点**        | **常用应用场景**       | **举例**                            |
| ----------- | --------------- | ------------------ | ---------------- | --------------------------------- |
| **数值 - 整数** | TINYINT         | 1 字节，范围小           | 状态码、性别、类型标记      | `0` (禁用), `1` (启用)                |
|             | SMALLINT        | 2 字节，范围中等          | 较小的计数、排序号、年龄     | `18` 岁, 排序 `100`                  |
|             | INT / INTEGER   | 4 字节，约正负 21 亿      | 用户 ID、文章 ID、普通计数 | 用户 ID: `10086`                    |
|             | BIGINT          | 8 字节，范围极大          | 分布式 ID、大系统交易流水号  | 雪花 ID: `142536789...`             |
| **数值 - 小数** | DECIMAL(M,D)*   | **精准小数**，无精度丢失     | 涉及钱、财务、精确计量      | 价格: `199.99`                      |
|             | FLOAT / DOUBLE  | 近似小数，可能存在微小误差      | 科学计算、地理经纬度、身高体重  | 纬度: `31.2304`                     |
| **字符串**     | VARCHAR(N)      | **可变长度**，省空间       | 用户名、地址、文章标题、邮箱   | `张三`, `test@qq.com`               |
|             | CHAR(N)         | **固定长度**，速度快       | 长度绝对固定的编码、密文     | 身份证、MD5密文、手机号                     |
|             | TEXT / LONGTEXT | 超长文本               | 文章正文、产品详情、系统日志   | 几千字的文章内容                          |
| **日期时间**    | DATETIME        | 8 字节，**不随时区改变**    | 业务约定的绝对时间        | 合同到期: `2026-12-31 23:59:59`       |
|             | TIMESTAMP       | 4 字节，**跟随系统时区改变**  | 数据自动记录的创建/修改时间   | 创建时间: `2026-06-08 15:30:00`       |
|             | DATE            | 只包含日期              | 生日、入职日期、活动当天     | `1998-10-01`                      |
|             | TIME            | 只包含时间              | 每日营业时间、闹钟        | `09:00:00`                        |
| **特殊类型**    | BOOLEAN         | 实际表现为 `TINYINT(1)` | 是/否 的逻辑判断        | `TRUE` (1), `FALSE` (0)           |
|             | JSON            | 存储结构化 JSON 数据      | 字段经常变动的配置信息、扩展属性 | `{"theme": "dark", "lang": "zh"}` |
